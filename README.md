# Website features

## Uitleg
Korte uitleg over hoe de straat en het volledige nieuwe “leen” systeem werkt.

## Registratie
De gebruiker kan registreren. Met dit account kan hij of zij dan abonnementen aanschaffen en / of beheren, zijn profiel bewerken en beheren, etc. We gebruiken hiervoor dezelfde database als van de webshop, zodat de gebruiker kan inloggen op beide websites met hetzelfde account.

## Abonnementen
De gebruiker heeft (na registratie) de mogelijkheid om abonnementen aan te schaffen en indien de gebruiker al een abonnement heeft, kan hij of zij dit upgraden naar een duurdere versie of downgraden naar een goedkopere versie.
